# Feature Graphic Guide

## Requirements
- **Size:** 1024 x 500 pixels
- **Format:** PNG or JPEG
- **Max File Size:** 1MB
- **Content:** App branding + tagline

## Quick Creation Methods

### Method 1: Canva (Recommended - Free & Easy)

1. **Go to Canva:** https://www.canva.com
2. **Create Custom Size:**
   - Click "Create a design"
   - Select "Custom size"
   - Enter: 1024 x 500 pixels
3. **Design Your Graphic:**
   - Add app icon/logo
   - Add tagline: "AI-Powered Travel Companion"
   - Use brand colors: Blue (#1976D2), Purple gradient
   - Add travel-related imagery (optional)
4. **Download:**
   - Click "Share" → "Download"
   - Format: PNG
   - Download

**Template Ideas:**
- Left: App icon
- Center: "Travel Buddy" text
- Right: "AI-Powered Travel Companion"
- Background: Gradient or travel image

### Method 2: Figma (Free)

1. **Go to Figma:** https://www.figma.com
2. **Create Frame:** 1024 x 500
3. **Design Elements:**
   - Import app icon
   - Add text layers
   - Apply gradients
4. **Export:** File → Export → PNG

### Method 3: Photoshop/GIMP

1. **New Document:** 1024 x 500 pixels
2. **Add Layers:**
   - Background (gradient or solid color)
   - App icon
   - Text: "Travel Buddy"
   - Tagline: "AI-Powered Travel Companion"
3. **Export:** Save as PNG

## Design Guidelines

### Colors
- Primary: #1976D2 (Blue)
- Secondary: #7C3AED (Purple)
- Background: White or gradient

### Typography
- App Name: Bold, 48-60px
- Tagline: Regular, 24-32px
- Font: Arial, Roboto, or similar

### Layout Options

**Option A - Centered:**
```
[        App Icon        ]
[     Travel Buddy       ]
[ AI-Powered Travel Companion ]
```

**Option B - Left-Right:**
```
[ Icon ]  Travel Buddy
          AI-Powered Travel Companion
```

**Option C - With Background:**
```
[Travel Background Image]
[  App Icon + Text Overlay  ]
```

## Validation Checklist

- [ ] Exactly 1024 x 500 pixels
- [ ] PNG or JPEG format
- [ ] Under 1MB file size
- [ ] App name visible
- [ ] Tagline included
- [ ] High quality (no pixelation)
- [ ] Readable text
- [ ] Professional appearance

## Common Mistakes to Avoid

❌ Wrong dimensions
❌ Text too small to read
❌ Low resolution/blurry
❌ Too much text
❌ Cluttered design
❌ File size over 1MB
