# Feature Graphic Design Guide

## Specifications
- **Size**: 1024 x 500 pixels
- **Format**: PNG or JPEG
- **Max file size**: 1MB
- **Content**: App logo + tagline "AI-Powered Travel Companion"

## Design Tools

### Option 1: Canva (Recommended - Easy)
1. Go to https://www.canva.com
2. Create custom size: 1024 x 500 px
3. Design elements:
   - Background: Gradient (blue #1976D2 to purple #7B1FA2)
   - App icon/logo in center or left
   - Text: "Travel Buddy" (large, bold, white)
   - Tagline: "AI-Powered Travel Companion" (smaller, white)
   - Optional: Travel-themed icons (plane, map, compass)
4. Export as PNG (high quality)

### Option 2: Figma (Professional)
1. Create new file with frame 1024 x 500
2. Add gradient background
3. Import app icon
4. Add text layers
5. Export as PNG 2x

### Option 3: Photoshop/GIMP
1. New document: 1024 x 500 px, 72 DPI
2. Create gradient background
3. Add logo and text
4. Save as PNG

## Design Tips
- Use high contrast (white text on dark/gradient background)
- Keep text readable at small sizes
- Center or left-align main elements
- Use app's brand colors (blue #1976D2, purple #7B1FA2)
- Avoid clutter - keep it simple and clean

## Quick Template
```
Background: Linear gradient (135deg, #1976D2 0%, #7B1FA2 100%)
Logo: Left side (200x200px area)
Title: "Travel Buddy" - 72pt, Bold, White
Tagline: "AI-Powered Travel Companion" - 32pt, Regular, White (80% opacity)
Icons: Small travel icons (plane, compass, map pin) scattered
```
